**主要语言**: TypeScript

## 🎯 项目背景

### 设计师的Cursor

Onlook被称为"设计师的Cursor"，它专为设计师打造，提供AI优先的设计工具，可以可视化地构建、样式化和编辑React应用。

### 核心特性

- 🎨 **可视化设计**: 可视化构建React应用
- 🤖 **AI优先**: AI驱动的设计工具
- ⚛️ **React集成**: 专为React应用设计
- 🔧 **实时编辑**: 实时样式编辑功能

## 🔍 技术分析

### 架构设计

```
onlook/
├── src/
│   ├── components/     # UI组件
│   ├── services/       # 服务层
│   ├── ai/            # AI功能
│   └── renderer/      # 渲染引擎
├── packages/          # 核心包
└── docs/             # 文档
```

### 核心功能

1. **可视化编辑器**
2. **AI设计助手**
3. **React代码生成**
4. **实时预览**

## 🚀 使用场景
